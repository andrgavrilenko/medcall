# MedCall

Одностраничный лендинг для вызова медицинского специалиста на дом. Один файл `index.html` — HTML + CSS + vanilla JS, без сборщиков и фреймворков.

## Архитектура

- Всё в `index.html`: стили в `<style>`, логика в `<script>`
- Отправка заявок через **EmailJS Browser SDK v4** (client-side, без бекенда)
- Деплой: GitHub Pages или Netlify (см. `DEPLOY.md`)

## Дизайн-токены

| Переменная | Значение | Назначение |
|---|---|---|
| `--green` | `#1B4F45` | Основной цвет (кнопки, акценты) |
| `--green-light` | `#2A6B5E` | Hover-состояние кнопок |
| `--bg` | `#F7F3EE` | Фон страницы |
| `--urgent` | `#E05A2B` | Срочный вызов, ошибки валидации |
| `--text` | `#2C2C2C` | Текст |
| `--muted` | `#7A7A7A` | Подписи, метки |
| `--border` | `#E8E3DC` | Границы полей |
| `--radius` | `16px` | Скругление карточек |

Шрифты: **Playfair Display** (заголовки), **Mulish** (текст) — Google Fonts.
Макс. ширина: 430px (mobile-first).

## Форма заявки

Обязательные поля: Услуга (select), Контактное лицо, Адрес.
Необязательные: Жалобы/помощь (textarea).
Тип вызова: обычный / срочный (toggle-кнопки).

EmailJS параметры шаблона: `{{urgency}}`, `{{service}}`, `{{contact_name}}`, `{{address}}`, `{{symptoms}}`, `{{time}}`.

## Правила при редактировании

- Весь код в одном файле — не разносить по отдельным .css/.js
- Стиль именования CSS: BEM-подобный через дефис (`.form-card`, `.step-num`)
- Анимации: классы `.fade-up` + `.delay-N`
- Валидация ошибок: красная подсветка `#E05A2B` на 2 сек, потом сброс
- При добавлении полей в форму — добавлять и в JS-валидацию, и в `templateParams`
- Язык интерфейса: русский
