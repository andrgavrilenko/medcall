## О проекте

MedCall — одностраничный лендинг для сервиса выезда врача на дом. Пользователь оставляет заявку через форму, заявка автоматически отправляется на email диспетчера через EmailJS. Никакого бекенда, никакой инфраструктуры — один HTML-файл.

## Возможности

- Форма заявки с валидацией обязательных полей
- Переключатель типа вызова: обычный / срочный
- Автоматическая отправка email при отправке формы
- Состояние успеха с подтверждением и временем прибытия
- Анимации появления элементов (fade-up)
- Адаптивная вёрстка, оптимизирована под мобильные устройства

## Стек

- **HTML + CSS + Vanilla JS** — без фреймворков и сборщиков
- **Google Fonts** — Playfair Display + Mulish
- **EmailJS Browser SDK v4** — отправка писем на стороне клиента
